# Fast Track – Full-Stack Technical Challenge

A **monorepo** project containing both **frontend** and **backend**.

## Tech Stack

### _General_

- **Node.js:** 22.21.1
- **Package Manager:** Yarn
- **Monorepo:** Frontend + Backend in a single repository

### _Backend_

- **Runtime:** Node.js
- **Framework:** Express
- **Port:** 3688

### _Frontend_

- **Framework:** React
- **Styling:** Tailwind CSS
- **Routing:** Simple React Router
- **Port:** 3614

### _Project Structure (Simplified)_

```bash
/root
 ├─ /backend     # Express API
 ├─ /frontend    # React application
 ├─ package.json
 └─ README.md
```

### _Getting Started_

```bash
# Clone the repository
git clone https://github.com/milenbubi/fasttrack.git

# Go into the project directory
cd fasttrack

# Install dependencies
yarn install

# Start the local development server and the frontend
yarn run dev
```

### The frontend application will be available at  [http://localhost:3614](    http://localhost:3614)
